# PDCA / CIPDA /ROMER User Guides and Notes

## Overview

This repository contains early version practical user guides, notes, examples, and supporting materials for applying the approach in real situations.

It is intentionally separate from the formal research and evaluation repositories (including IJQRM outputs).

Those repositories examine evidence.

This repository exists to encourage use.

## Purpose

The objective is simple:

* Make the methods available.
* Allow people to try them.
* Encourage adaptation and extension.
* Learn from practical application.

The assumption is straightforward:

Methods become valuable when they are repeatedly used, tested, challenged, and improved in practice.

## Contents

* User guides
* Quick-start documents
* Primers
* Worked examples
* Notes and observations
* Templates and supporting materials

## Using This Repository

Take what is useful.

Apply it to real work.

Adapt where necessary.

Record what you learn.

Share improvements where appropriate.

No certification, endorsement, or permission process is required for experimentation and normal use.
